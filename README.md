 Я решил использовать следующие технологии:\
`React`\
`Redux/ReduxToolkit` - для инкапсуляции состояния от представления\
`TypeScript` - для строгой типизации состояния и компонентов React\
`React-hook-form` - для создания компонента формы\
`Bootstrap` - для стилизации\


### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
